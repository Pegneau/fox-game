# Fox Game Project

This is my personal repository for the Fox Game project. Inside, you'll find my implementation, code progression, and local experiments building this virtual pet game.

## Tech Stack & Features

- Vanishing/state-driven animations
- Custom game loop and state machine
- CSS Grid/Flexbox UI layout
